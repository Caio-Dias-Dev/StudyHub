# 📚 StudyHub

O **StudyHub** é uma plataforma web desenvolvida para ajudar estudantes a organizarem sua rotina de estudos de forma prática, intuitiva e eficiente. O projeto foi criado com foco em produtividade, permitindo o gerenciamento de tarefas, acompanhamento do progresso e planejamento diário dos estudos em uma interface moderna e responsiva.

## ✨ Funcionalidades

* 📅 Calendário para organização da rotina de estudos.
* ✅ Gerenciamento de tarefas e metas diárias.
* 📊 Barra de progresso para acompanhar o desempenho.
* ⏱️ Controle de horas de estudo.
* 🎯 Organização por disciplinas.
* 💾 Armazenamento local dos dados utilizando **LocalStorage**.
* 🌙 Interface moderna e responsiva.

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (ES6+)

## 🎯 Objetivo

O principal objetivo do StudyHub é oferecer uma ferramenta simples e eficiente para auxiliar estudantes no planejamento de seus estudos, incentivando a organização, a disciplina e o acompanhamento da evolução diária.
